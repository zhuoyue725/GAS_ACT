# GAS Attach Editor - Unreal Engine Plugin
类似UE 命令行ShowDebug AbilitySystem的图形化编辑器功能，方便查看

### 与官方Debug信息对比
优点：
- 支持专用服务器的数据查看
- 支持激活/未激活的筛选
- 便捷快速查找便于其他调试

### 在非Shippng模式下使用在命令行里输入`GASAttachEditorShow`的命令
